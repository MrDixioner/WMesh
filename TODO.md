# Progress
**1.7.1**
**RUS**
- В настройках плагина заменено его название с W_Mesh на WMesh.
- В именах файлов примитивов убраны подчёркивания, а так же в именах иконок убраны цифры "64".
- Примитив Spiral (Спираль) заменён на Spring (Пружина).
- Заменены все иконки на более читаемые.

**ENG**
- The plugin's name in the settings has been changed from W_Mesh to WMesh.
- Underscores have been removed from primitive file names, and the number "64" has been removed from icon names.
- The Spiral primitive has been replaced with Spring.
- All icons have been replaced with more readable ones.

**1.7**
**RUS**
- [x] Добавлен объект Спираль (**Spiral**) - Настройка основного диаметра, диаметра верха и диаметра основания, Высота спирали и Количество витков, настройка Сегментов спирали и сечения, а так же Центровка спирали и Коэффициент усадки.

**ENG**
- [x] Added a Spiral object (**Spiral**) - Adjustments for the Main diameter, Top diameter, and Base diameter, Spiral height and Number of turns, adjustments for Spiral segments and Cross-sections, as well as Spiral centering and Shrinkage factor.
- [x] Changed primitive icons (further refinement is planned).
